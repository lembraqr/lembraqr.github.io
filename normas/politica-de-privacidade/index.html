Política de Privacidade

Bem-vindo à Política de Privacidade do LembraQr. Este documento descreve como coletamos, usamos, armazenamos e protegemos os dados pessoais dos nossos usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018). Nosso objetivo é garantir transparência e segurança no tratamento de seus dados. Leia atentamente antes de utilizar nossos serviços.

  1. Quem Somos
    O LembraQr é uma plataforma que oferece produtos físicos, como QR codes em adesivos, placas ou outros formatos, integrados a funcionalidades digitais, como memoriais online personalizados. Nosso propósito é permitir que os usuários criem e acessem memoriais digitais de forma prática e segura, vinculados a produtos físicos, para preservar memórias e histórias.
  2. Quais Dados Coletamos
    Para oferecer nossos serviços, coletamos os seguintes tipos de dados pessoais:
    • Dados Cadastrais: Nome, e-mail, telefone e outros dados de contato fornecidos durante o cadastro ou aquisição de produtos.
    • Dados de Pagamento: Informações necessárias para processar transações, como dados de cartão de crédito ou outros métodos de pagamento, coletados por plataformas de pagamento seguras.
    • Conteúdo do Usuário: Textos, imagens, vídeos ou outros materiais enviados pelos usuários para personalização de memoriais online.
    • Dados de Navegação: Informações coletadas durante o uso do site, como endereço IP, tipo de dispositivo, navegador, páginas visitadas, tempo de navegação e interações com a plataforma.
  3. Como Coletamos os Dados
    Os dados são coletados das seguintes formas:
    • Por meio de formulários preenchidos voluntariamente pelo usuário no site ou aplicativo, como durante o cadastro ou criação de memoriais.
    • Durante o processo de compra de produtos físicos ou assinatura de serviços digitais.
    • Automaticamente, por meio de cookies e tecnologias de rastreamento, ao acessar o site ou utilizar os serviços digitais.
    • Através do conteúdo enviado pelo usuário para personalização dos memoriais online.
  4. Finalidade da Coleta de Dados
    Os dados coletados são utilizados para as seguintes finalidades:
    • Cadastro e autenticação de usuários na plataforma.
    • Criação, personalização e manutenção de memoriais online.
    • Gestão de pedidos de produtos físicos e serviços digitais.
    • Processamento de pagamentos de forma segura.
    • Comunicação com os usuários, incluindo envio de notificações sobre pedidos, atualizações de serviços ou suporte técnico.
    • Melhoria contínua dos serviços, por meio de análises de uso e desempenho da plataforma.
    • Cumprimento de obrigações legais ou regulatórias.
  5. Base Legal para o Tratamento de Dados
    O tratamento de dados pessoais pelo LembraQr é baseado nas seguintes hipóteses previstas na LGPD:
    • Consentimento: Para coleta de dados fornecidos voluntariamente, como conteúdo de memoriais ou dados cadastrais, quando o usuário expressamente concorda.
    • Execução de Contrato: Para processar pedidos, gerenciar contas ou fornecer acesso a serviços digitais contratados.
    • Obrigação Legal: Para cumprir exigências legais, como emissão de notas fiscais ou resposta a autoridades.
    • Legítimo Interesse: Para análises de navegação e melhoria dos serviços, desde que não prejudiquem os direitos dos titulares.
  6. Compartilhamento de Dados
    O LembraQr pode compartilhar dados pessoais com terceiros nas seguintes situações:
    • Com plataformas de pagamento, para processar transações de forma segura.
    • Com provedores de serviços de hospedagem e armazenamento em nuvem, para garantir o funcionamento dos memoriais online.
    • Com ferramentas de análise (como Google Analytics), para monitorar o desempenho do site, utilizando dados anonimizados sempre que possível.
    • Com autoridades competentes, quando exigido por lei ou para proteger direitos legais.
    Todo compartilhamento é realizado com contratos que garantem a proteção dos dados e limitam seu uso às finalidades específicas.
  7. Segurança dos Dados
    Adotamos medidas técnicas e organizacionais para proteger os dados pessoais, incluindo:
    • Criptografia AES-256 para armazenamento de dados sensíveis.
    • Protocolo HTTPS para comunicação segura entre o usuário e nossos servidores.
    • Firewalls e sistemas de detecção de intrusão para proteger nossos servidores.
    • Políticas internas de acesso restrito aos dados por funcionários autorizados.
    Apesar dessas medidas, nenhum sistema é completamente imune a riscos. Em caso de incidentes de segurança, os usuários afetados serão notificados, conforme exigido pela LGPD.
  8. Direitos do Titular dos Dados
    Conforme a LGPD, o usuário tem os seguintes direitos em relação aos seus dados pessoais:
    • Confirmação e Acesso: Verificar se tratamos seus dados e acessar os dados armazenados.
    • Correção: Solicitar a atualização ou correção de dados incompletos, inexatos ou desatualizados.
    • Exclusão: Solicitar a exclusão de dados desnecessários, excessivos ou tratados em desconformidade com a LGPD.
    • Portabilidade: Solicitar a transferência de seus dados para outro fornecedor de serviços, quando tecnicamente viável.
    • Revogação de Consentimento: Retirar o consentimento previamente dado, quando aplicável.
    • Oposição: Opor-se ao tratamento de dados com base em legítimo interesse, em caso de irregularidades.
    • Informação sobre Compartilhamento: Saber com quais entidades seus dados foram compartilhados.
    Para exercer esses direitos, entre em contato pelo e-mail support@lembraqr.com. Responderemos às solicitações no prazo legal.
  9. Uso de Cookies e Tecnologias de Rastreamento
    Utilizamos cookies e tecnologias similares para melhorar a experiência do usuário e analisar o desempenho do site. Os tipos de cookies incluem:
    • Cookies Funcionais: Necessários para o funcionamento do site, como autenticação de login e manutenção de sessões.
    • Cookies de Desempenho: Coletam dados anônimos sobre o uso do site para melhorar sua funcionalidade.
    • Cookies de Publicidade: Usados para personalizar anúncios, quando aplicável, com base em interesses do usuário.
    O usuário pode gerenciar preferências de cookies nas configurações do navegador, mas a desativação de cookies funcionais pode comprometer o uso do site.
  10. Tempo de Retenção dos Dados
    Os dados pessoais são retidos pelo tempo necessário para cumprir as finalidades descritas, conforme a seguir:
    • Dados Cadastrais: Mantidos enquanto a conta do usuário estiver ativa ou pelo prazo necessário para cumprir obrigações legais (ex.: 5 anos para notas fiscais).
    • Conteúdo do Usuário: Armazenado enquanto o memorial online estiver ativo. Após o encerramento da conta ou inatividade por 12 meses, os dados serão excluídos ou anonimizados, salvo se exigido por lei.
    • Dados de Navegação: Retidos por até 6 meses, sendo anonimizados após esse período, exceto quando exigido por lei.
    Após o término do período de retenção, os dados serão excluídos ou anonimizados de forma segura.
  11. Responsável pelo Tratamento de Dados
    O LembraQr é o controlador dos dados pessoais tratados no âmbito dos nossos serviços. Para dúvidas, solicitações ou reclamações, entre em contato com nosso Encarregado de Proteção de Dados (DPO) pelo e-mail: support@lembraqr.com.
  12. Alterações na Política de Privacidade
    Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças em nossas práticas ou na legislação aplicável. A versão atualizada será publicada nesta página, e os usuários serão notificados por e-mail ou pelo site, quando necessário. Recomendamos revisar esta política regularmente. O uso continuado dos serviços após alterações constitui aceitação das novas condições.
  13. Lei Aplicável
    Esta Política é regida pelas leis da República do Brasil, em especial a LGPD. Disputas relacionadas serão resolvidas preferencialmente por meios extrajudiciais, como mediação, ou, se necessário, pelos tribunais competentes de Campo Mourão, PR.
